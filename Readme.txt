Ao usar NetBeans

1º - Extrair o projeto para a pasta \NetBeansProjects, que normalmente está nos Documentos do user
2º - Para executar este programa é necessário JavaFX*
3º - No NetBeans do lado direito nos projetos, vamos às propriedades do nosso projeto
4º - Ir a Actions, e selecionamos "Run Project"
5º - Execute Goals: javafx:run (temos de colocar assim, para garantir que é executado)
6º - Executar o projeto.

*JavaFX no NeBeans
Tools--> Options--> Java--> Maven--> Manage Java Plataforms--> Add Plataform--> Download OpenJDK--> Distribution--> ZULU
